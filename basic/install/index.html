Manual installation

  • From Eclipse, select ‘Help’ -> ‘Eclipse Marketplace…’.
  • In the ‘Find’ field, type ‘Claret’ and hit ‘Go’ button.
  • Hit install button.
  • Accept the terms of the license agreement then ‘Finish’.
  • Restart Eclipse.
